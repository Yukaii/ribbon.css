# ribbon.css

Retro Office 2007 ribbon component styles for React.

## Install

```bash
npm install ribbon.css
```

## Usage

```jsx
import { RetroRibbon } from "ribbon.css";
import "ribbon.css/ribbon.css";

export default function App() {
  return <RetroRibbon title="Retro Ribbon UI Reference Canvas" />;
}
```

## Exports

- `RetroRibbon`
- `defaultTabs`
- `defaultGroups`
- `src/ribbon.css`
